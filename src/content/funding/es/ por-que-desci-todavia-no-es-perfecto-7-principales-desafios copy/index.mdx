## OpenScience Fund


### Acerca del fondo

OpenScience Fund es una iniciativa dedicada a apoyar proyectos que promueven la ciencia abierta y el desarrollo de software de código abierto para la investigación científica.

### Características principales

- **Monto de financiamiento**: Otorga subvenciones de hasta $100,000 por proyecto.
- **Tipos de proyectos**: Se enfoca en investigación científica y desarrollo de software para ciencia.
- **Proceso de aplicación**: Convocatorias abiertas trimestralmente.
- **Requisitos clave**: Los proyectos deben ser de código abierto y promover prácticas de ciencia abierta.

### Proyectos destacados

1. **Proyecto Galaxy**: Una plataforma de análisis de datos genómicos de código abierto.
2. **OpenNeuro**: Un repositorio de datos de neuroimagen de acceso abierto.

### Cómo aplicar

1. Visita el [sitio web oficial](https://openscience.fund).
2. Revisa las guías de aplicación y criterios de evaluación.
3. Prepara tu propuesta de proyecto según las directrices proporcionadas.
4. Envía tu aplicación durante el período de convocatoria abierta.

Para más información y para mantenerte actualizado sobre las próximas convocatorias, síguenos en [Twitter](https://twitter.com/opensciencefund) y únete a nuestra [comunidad en Discord](https://discord.gg/opensciencefund).
